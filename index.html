Seam carving

Implementation of a seam carving algorithm for content aware image resizing

Download this project as a .zip file Download this project as a tar.gz file

User interface and features

  • Inserting and removing vertical seams (this affects the width of the image)
  • Visualising the energy, the accumulated energy and seams
  • The option to use forward energy while calculating the accumulated energy

Note: This was written for a school assignment and it's far from optimal and still lacks a lot of features. Do not expect high performance or high quality.
